# Codex GAS 스크립트

## 운영 주의사항
- **ContactsApp는 종료(Deprecated)** 되었습니다. 해당 환경에서는 연락처 동기화/점검이 자동으로 스킵되며, People API로 이전이 필요합니다.
- **Drive 체크는 DriveApp 권한이 필요**합니다. 권한이 없으면 드라이브 체크가 즉시 중단됩니다.

## 머지 충돌 시 필수 확인
- `<<<<<<<`, `=======`, `>>>>>>>` 표시가 남아 있으면 **스크립트가 실행되지 않습니다.**
